# Smart Workforce Analytics Management System

## System Architecture

This project follows Object-Oriented Programming principles.

Classes Used:

1. Employee
   - Represents employee data

2. DatabaseManager
   - Handles SQLite database operations
   - CRUD operations

3. AnalyticsEngine
   - Performs data analysis using Pandas and NumPy

4. ReportGenerator
   - Generates visualizations using Matplotlib

## Technologies Used

- Python
- SQLite
- NumPy
- Pandas
- Matplotlib
- Object-Oriented Programming
